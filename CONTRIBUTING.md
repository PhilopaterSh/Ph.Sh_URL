# Contributing to Ph.Sh_URL

First off, thank you for considering contributing to Ph.Sh_URL! It's people like you that make open source such a great community.

## How Can I Contribute?

### Reporting Bugs
If you find a bug, please make sure to report it by opening a new issue. Please include as much detail as possible, including:
- A clear and descriptive title.
- Steps to reproduce the bug.
- The version of the tool you are using.
- Any relevant logs or screenshots.

### Suggesting Enhancements
If you have an idea for a new feature or an enhancement to an existing one, feel free to open an issue to discuss it. This allows us to coordinate and make sure your idea aligns with the project's goals.

### Pull Requests
We welcome pull requests! If you'd like to contribute code, please follow these steps:

1.  **Fork the repository** and create your branch from `main`.
2.  **Set up your development environment**: Make sure you have Go installed (version 1.25 or higher).
3.  **Make your changes**: Write your code and any accompanying tests.
4.  **Ensure your code lints**: Run `go fmt ./...` to format your code.
5.  **Submit a pull request**:
    - Use a clear and descriptive title.
    - Provide a summary of the changes in the pull request description.
    - Reference any related issues.

We will review your pull request as soon as possible. Thank you for your contribution!
